# Installation

To install Fiori Role Testing execute the following steps:

1. [Obtain and import `Fiori Role Testing` transport files](installation/step-1.md)
2. [Activate `zftappusagerep`, `zfaadmin` and `zftappusageplug` frontend ICF nodes](installation/step-2.md)
3. [Enable `ZNYPEFACEN_SRV` and `ZNYPEFAMAN_SRV` backend odata services](installation/step-3.md)
4. [Assign `ZNYPE_FAMAN` and `ZNYPE_FAMEM` pfcg roles](installation/step-4.md) 
5. [Set the parameters for activation key and log mode](installation/step-5.md)
